5.1 年間熱負荷係数（PAL*）
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

年間熱負荷係数（PAL*）の基準は次の式を満たす場合を「達成」とし、満たさない場合を「非達成」とする。

.. math::
    :nowrap:

    \begin{align*}
        f_{PAL*, dsn} \le f_{PAL*, std} \tag{5.1}
    \end{align*}

:math:`f_{PAL*, dsn}` : 年間熱負荷係数（PAL*）の設計値, MJ / ( |m2| ･ 年)

:math:`f_{PAL*, std}` : 年間熱負荷係数（PAL*）の基準値, MJ / ( |m2| ･ 年)

5.6 設計一次エネルギー消費量（建築物エネルギー消費性能基準）
